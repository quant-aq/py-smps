smps.fit.LogNormalFitResults¶

class smps.fit.LogNormalFitResults(params, error_matrix, fittedvalues, modes, **kwargs)¶

LogNormalFitResults holds the results produced by LogNormal.

Parameters

  • params (array) – The parameters of the fitted model.

  • error_matrix (array) – The one standard deviation errors associaed with the paramters, given in the same order as params.

  • fittedvalues (array) – The results from evaluating the model at each of the independent variables in the training data.

  • modes ({1,2,3}) – The number of modes the model has.
