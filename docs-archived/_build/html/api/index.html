Utility Functions¶

Utility Functions

smps.utils.make_bins(**kwargs)

'make_bins` computes the upper and lower bounds and midpoints of all the bins given either * boundaries * boundaries_left and boundaries_right * or lb, ub, and midpoints (channels_per_decade optional).

Models¶

Particle Sizer Models

smps.models.GenericParticleSizer(data, bins, ...)

GenericParticleSizer is the base class for all available particle sizing instruments.

smps.models.SMPS(**kwargs)

smps.models.Grimm11D(**kwargs)

The Grimm11D is an optical particle counter that uses a blue laser to count and size individual particles.

smps.models.POPS(**kwargs)

The Portable Optical Particle Spectrometer is based on the design of R.

smps.models.ParticlesPlus(**kwargs)

The Particles Plus is a consumer-grade optical particle counter that uses a red laser to count and size individual particles.

smps.models.AlphasenseOPCN2(**kwargs)

The Alphasense OPC-N2 is a consumer-grade optical particle counter that uses a 658 nm red laser to count and size individual particles.

smps.models.AlphasenseOPCN3(**kwargs)

The Alphasense OPC-N3 is a consumer-grade optical particle counter that uses a 658 nm red laser to count and size individual particles.

smps.models.Modulair(**kwargs)

QuantAQ's MODULAIR sensors use the Alphasense OPC-N3, a consumer-grade optical particle counter that uses a 658 nm red laser to count and size individual particles.

smps.models.ModulairPM(**kwargs)

QuantAQ's MODULAIR-PM sensors use the Alphasense OPC-N3, a consumer-grade optical particle counter that uses a 658 nm red laser to count and size individual particles.
