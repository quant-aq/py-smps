smps.io.smps_from_txt(fpath, column=True, delimiter=',', as_dict=True, **kwargs)¶

Read an SMPS txt file as exported by the TSI AIM software.

Parameters

  • fpath (string) – The file path to txt file to be read.

  • column (bool) – If your data is in ‘column’ format, set True. Otherwise, set False, defaults to True.

The smps.utils Module¶

smps.utils.make_bins(**kwargs)¶

‘make_bins` computes the upper and lower bounds and midpoints of all the bins given either * boundaries * boundaries_left and boundaries_right * or lb, ub, and midpoints (channels_per_decade optional).

Parameters

  • boundaries (list of lists, or numpy array) – A list containing, for each bin, a list of the lower and upper bound for the bin.

  • boundaries_left (list) – A list of lower boundaries for each bin.

  • boundaries_right (list) – A list of upper boundaries for each bin.

  • lb (float) – The lower bound of the lowermost bin.

  • ub (float) – The upper bound of the uppermost bin.

  • midpoints (list) – A list of midpoints for each bin.

  • channels_per_decade (float) – A measure of bin width on the log scale.

  • mean_calc ({"gm", "am"}) – Determins wether the geometric or arethmetic mean should be used to compute the midpoints.

Returns

For each bin a row is given with the lower boundary, the midpoint, and the upper bounary for that bin.

Return type

numpy array

The GenericParticleSizer class¶

class smps.GenericParticleSizer(data, bins, **kwargs)¶

Bases: object

GenericParticleSizer is the base class for all available particle sizing instruments. It contains all of the basic functionality and methods used for making calculations.

It can receive data in several formats including: dN, dNdlogDp.

Parameters

  • data (DataFrame) – Data values indexed by timestamp with bins as columns.

  • bins (list of lists) – A list, ordered respective to the bins, where for each bin a min, midpoint, and max particle size is given.

  • dp_units ({'um','nm'}) – Units in which the diameter is measured, defaults to ‘um’.

stats(weight='number', dmin=0.0, dmax=1000.0, rho=1.65, **kwargs)¶

Calculate and return the total number of particles, total surface area, total volume, and total mass between dmin and dmax. In addition, the arithmetic mean (AM), geometric mean (GM), mode (Mode), and geometric standard deviation (GSD, CMD) are calculated and are weighted by the input parameter weight.

Parameters

  • weight ({"number", "surface", "volume", "mass"}) – The dimension to consider in computing statistics.

  • dmin (float) – The minimum particle diameter in microns, defaults to 0.

  • dmax (float) – The maximum particle diameter in microns, defaults to 1e3.

  • rho (float) – The particle density in units of g/cc3, defaults to 1.65.

Returns

The statistics for each timestamp.

Return type

DataFrame

The LogNormal class¶

class smps.fit.LogNormal¶

Bases: object

LogNormal uses non-linear least squares to fit a log-normal model, with up to three modes.

fit(X, Y, modes=1, xmin=None, xmax=None, weight='number', fit_kwargs=None, **kwargs)¶

fit uses non-linear least squares to fit a log-normal model, with up to three modes.

Parameters

  • X (array-like) – The independant variable.

  • Y (array-like) – The dependent variable.

  • modes ({1,2,3}) – Number of modes for the model to have, defaults to 1.

  • xmin (float, optional) – Lower bound by which to crop data.

  • xmax (float, optional) – Upperbound by which to crop data.

  • weight ({"number", "surface", "volume"}) – Dimension of interest.

  • fit_kwargs (dict, optional) – Options to be passed to scipy.optimize.curve_fit. Read more here.

  • p0 (list, optional) – Initial guesses passed to scipy. optimize.curve_fit for the parameters. Must have length 3*`modes`.

  • bounds (2-tuple) – Bounds to be passed to scipy. optimize.curve_fit. Defaults to (0, [1e9, 1e5, 5]*self.modes).

Returns

A fitted log-normal model.

Return type

LogNormalFitResults object
