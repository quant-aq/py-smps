smps.plots.heatmap¶

smps.plots.heatmap(X, Y, Z, ax=None, logy=True, cbar=True, hide_low=True, cmap='viridis', fig_kws={}, cbar_kws={}, plot_kws={}, **kwargs)¶

Plot the heatmap of the particle size distribution. All NaN’d values will be converted to zeroes.

Parameters

  • X (array-like) – The boundaries of cells in x-axis dimension.

  • Y (array-like) – The boundaries of cells in y-axis dimension.

  • Z (2D array-like) – The color value in the cells.

  • ax (matplotlib.pyplot Axes) – A matplotlib.pyplot Axes in which to build the plot. If None, a new Figure is instantiated. Defaults to None.

  • logy (bool) – Set whether the y axis is on a log scale, defaults to True.

  • cbar (bool) – Set whether to include a color bar in the plot key, defaults to True.

  • hide_low (bool) – Set whether Z values below cbar_min should be set to cbar_min, defaults to True.

  • cmap (string or Colormap) – A color map for matplotlib.pyplot pcolormesh, defaults to default_cmap. See here.

  • fig_kws (dict) – Options for the matplotlib.pyplot figure. See here, efaults to dict(figsize=(16,8).

  • cbar_kws (dict) – Options for the matplotlib.pyplot colorbar. See here.

  • plot_kws (dict) – Options for the matplotlib.pyplot Axes. See here <https://matplotlib.org/stable/api/_as_gen /matplotlib.pyplot.gca.html>’__, Defaults to `dict(alpha=1, edgecolor=None, linewidth=0).

  • cbar_min (float) – The minimum value for the color bar, defaults to the smallest Z value (unless it is not greater than zero, and then it will default to one).

  • cbar_max (float) – The maximum value for the color bar, defaults to the largest Z value.

Returns

A plot of the histogram.

Return type

matplotlib.pyplot Axes
