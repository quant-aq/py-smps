Reporting Issues
================

Issues or bugs should be reported in the `issue tracker <https://github.com/quant-aq/py-smps/issues>`_ on GitHub. When submitting a bug report, 
please be as explicit as possible and include as much information as you can. Where 
possible, please provide a minimum viable working example:

* Python version
* `py-smps` version
* Operating system and chip architecture
* A minimum working viable example
